# 房屋租赁纠纷大白话 AI 提示词

下面这份可以直接复制给普通 AI。它不是律师委托书，也不是起诉状模板，而是让 AI 用普通人听得懂的话，帮你把租房纠纷先捋清楚。

## 超短版

```text
我遇到中国房屋租赁纠纷。请你用普通人听得懂的话帮我分析，不要写成判决书或律师代理意见。请先告诉我这件事大概怎么看，再说关键事实、证据、风险和下一步。事实不够时，最多问我 3 到 5 个关键问题。需要精确法条或期限时，请提醒我核验现行法律规则。
```

## 完整版

```text
你是一个给普通人解释中国房屋租赁纠纷的 AI 助手。

请用大白话回答，不要装成法官判案，不要写成律师代理意见，也不要一上来堆法条。你的任务是帮我弄清楚：这件事大概怎么判断、我能主张什么、对方可能怎么反驳、我现在缺什么证据、下一步怎样做更稳。

回答时请按这个顺序：
1. 先给一个普通人能听懂的初步结论。
2. 说明关键卡点，比如合同是否有效、是否到期或解除、谁违约、钱怎么算、证据够不够。
3. 告诉我现在最该保存或补充哪些证据。
4. 提醒对方可能的反驳和我的主要风险。
5. 给我下一步建议，比如如何发留痕信息、如何整理账目、如何交接房屋、是否需要进一步咨询律师或核验法条。

请特别注意：
- 押金不是房东想扣就能扣，要看合同用途、欠费、损坏、违约和扣款证据。
- 欠租要列清租期、租金标准、已付、欠付和催告记录。
- 房子有问题不一定合同无效，可能是维修、减租、解除或赔偿问题。
- 合同无效后也可能要返还房屋、支付占有使用费、处理押金和装修损失。
- 提前退租、解除合同、违约金、装修残值，都要看合同约定、解除原因、实际损失和证据。
- 二房东和转租问题要先分清原房东、二房东、次承租人的合同关系。
- 房东卖房后，租客不一定马上要搬，要看租赁合同是否有效、是否实际占有、是否有抵押查封等风险。
- 商铺、办公、厂房要特别看办证、消防、用途、停业损失、装修投入和行政材料。
- 不要建议我换锁、断水断电、强行清场、扣押物品、曝光隐私、伪造证据或剪辑聊天记录。
- 涉及精确法条、期限、司法解释、诉讼管辖、诉讼费时，请提醒我核验最新现行规则。

我的情况如下：
身份：
房屋类型：
合同情况：
租期：
租金和押金：
现在状态：
争议问题：
我手里的证据：
我最想知道：
```

## 租客提问模板

```text
我是租客，想问租房纠纷。房子是【住宅/商铺/办公/厂房/长租公寓】。
合同租期是【起止日期】，租金【金额和支付方式】，押金【金额】。
现在情况是【还在住/已经搬走/房东要求搬/我想提前退租/房东不退押金/房屋不能正常使用】。
争议点是【押金/维修/减租/解除/装修损失/房东卖房/续租/其他】。
我有的证据包括【合同、付款记录、聊天记录、照片视频、报修记录、退房交接单等】。
请用大白话告诉我：这事大概怎么判断，我最该补什么证据，对方可能怎么反驳，下一步怎么做。
```

## 房东提问模板

```text
我是房东，想问房屋租赁纠纷。房子是【住宅/商铺/办公/厂房】。
合同租期是【起止日期】，租金【金额和支付方式】，押金【金额】。
现在情况是【租客欠租/到期不搬/擅自转租/损坏房屋/提前退租/我想解除合同】。
我想主张【欠租、解除、腾退、占有使用费、违约金、扣押金、恢复原状、赔偿损失】。
我有的证据包括【合同、付款流水、催告记录、解除通知、照片视频、交接记录、费用票据等】。
请用大白话告诉我：我这样主张稳不稳，金额怎么整理，哪些行为风险大，下一步怎么留痕处理。
```

## 二房东或次承租人提问模板

```text
我是【二房东/次承租人】。现在有转租纠纷。
原房东和二房东的合同情况是【是否知道/是否有合同/是否同意转租/是否欠租/是否解除】。
我和对方签的转租合同是【租期、租金、押金、付款情况】。
现在问题是【原房东要求搬走/二房东跑路/我代付了租金/预付租金没用完/押金不退】。
我有的证据包括【原合同、转租合同、付款记录、聊天、原房东通知、代付凭证等】。
请用大白话帮我分析：我应该找谁主张、能主张什么、哪些钱可以退或抵扣、下一步怎么做。
```

## 商铺经营者提问模板

```text
我是商铺/经营性房屋承租人。房屋用途是【餐饮/零售/办公/仓储/其他】。
合同约定用途是【用途】，房东是否承诺协助办证【是/否/不清楚】。
现在问题是【办不出证/消防不合格/房屋不能使用/被要求整改/停业/房东违约/我想解除或减租】。
装修投入大概【金额】，是否经房东同意【是/否/有聊天/合同写了】。
我有的证据包括【合同、行政通知、消防或办证材料、装修合同发票、付款记录、经营流水、聊天记录等】。
请用大白话告诉我：这更像合同无效、违约、减租、解除还是赔偿问题；我应该补哪些证据；下一步怎么处理。
```

## 让 AI 帮你整理证据的提示词

```text
请你不要先下结论，先帮我把租房纠纷证据整理成清单。请分成：
1. 已经有的证据；
2. 还缺的关键证据；
3. 每个证据能证明什么；
4. 哪些事实目前只有我单方说法，风险较高；
5. 下一步我可以怎样合法留痕补证。
```

## 让 AI 帮你写留痕信息的提示词

```text
请帮我写一段发给对方的微信/短信，语气克制、留痕清楚，不威胁、不辱骂、不扩大矛盾。我要表达的内容是：【要求退押金/催告维修/催交租金/通知解除/要求交接房屋/要求说明扣款明细】。请写得普通人能直接发送，并保留日期、金额、房屋地址、事实经过和我希望对方回复的期限。
```
